# Справочник переменных Open-Meteo API

Диátaxis-тип: **Reference** — точная информация, без обучения.

## Почасовые переменные (hourly)

| Переменная | Единица | Описание |
|:-----------|:--------|:---------|
| `temperature_2m` | °C | Температура на высоте 2 м |
| `apparent_temperature` | °C | Ощущаемая температура |
| `relative_humidity_2m` | % | Относительная влажность |
| `dew_point_2m` | °C | Точка росы |
| `precipitation` | мм | Осадки за час |
| `precipitation_probability` | % | Вероятность осадков |
| `weather_code` | WMO | Код погодных условий |
| `wind_speed_10m` | м/с | Скорость ветра на 10 м |
| `wind_direction_10m` | ° | Направление ветра |
| `wind_gusts_10m` | м/с | Порывы ветра |

## Конфигурация проекта (`src/weather_dashboard/config.py`)

### Пороги предупреждений (ThresholdConfig)

| Поле | Дефолт | Описание |
|:-----|:-------|:---------|
| `wind_gust_ms` | 15.0 м/с | Порог порывов ветра |
| `precip_prob_pct` | 70.0 % | Порог вероятности осадков |
| `precip_sum_mm` | 10.0 мм | Порог суммы осадков |
| `frost_temp_c` | −10.0 °C | Порог мороза |
| `heat_temp_c` | 35.0 °C | Порог жары |

### Параметры окна прогулки (WalkWindowConfig)

| Поле | Дефолт | Описание |
|:-----|:-------|:---------|
| `precip_prob_max_pct` | 30.0 % | Макс. вероятность осадков |
| `wind_speed_max_ms` | 10.0 м/с | Макс. скорость ветра |
| `temp_min_c` | −5.0 °C | Мин. температура |
| `temp_max_c` | 30.0 °C | Макс. температура |
| `horizon_hours` | 48 | Горизонт поиска в часах |

### Коэффициенты ComfortIndex (ComfortIndexConfig)

| Поле | Дефолт | Описание |
|:-----|:-------|:---------|
| `t_opt` | 22.0 °C | Оптимальная ощущаемая температура |
| `rh_ok` | 60.0 % | Порог влажности дискомфорта |
| `ws_ok` | 5.0 м/с | Порог ветра дискомфорта |
| `k_t` | 2.0 | Коэффициент температуры |
| `k_h` | 0.5 | Коэффициент влажности |
| `k_w` | 1.5 | Коэффициент ветра |
| `k_p` | 30.0 | Коэффициент осадков |

---

## Формула ComfortIndex

CI = 100 − k_t · |AT − t_opt| − k_h · max(0, RH − rh_ok) − k_w · max(0, WS − ws_ok) − k_p · PP / 100
CI = clamp(CI, 0, 100)

| Категория | Диапазон |
|:----------|:---------|
| Комфортно | CI ≥ 70 |
| Терпимо | 40 ≤ CI < 70 |
| Тяжело | CI < 40 |
